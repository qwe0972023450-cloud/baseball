
# BaseballAgent Manager — v1.1 (整合版)

已依照需求完成：
- 客戶列表只顯示「姓名 / 所屬球隊 / 薪資 / 評價」，可展開查看詳情與本季數據。
- 例行賽長度 40–45 週（隨機），之後自動進入季後賽並產生各聯盟冠軍（美聯 / 國聯），並在「冠軍」頁面可查。
- 球員會依每週評分提升能力（受潛力上限限制）；球探（潛力）= 可能到達的最高層級。
- 評價規則：
  - 平均評價 < 2 連續 4 週 → 解約
  - 評價 3–4 → 讓渡名單
  - 5–6 一般球員 / 7–8 主力球員 / 8–9 當家球星 / 9–10 頂尖球星
- 退休年紀 35–45 歲：每年機率遞增，於週期檢查觸發。
- 新增「每週棒球新聞」📰：自動產出最佳/最差球員新聞，以報紙風格呈現。
- 移除女性名字（names.js 男性化），並加入 `js/data/players_mlb.js`（可切換成真實球員資料來源）。

部署：直接打開 index.html 即可離線執行。

擴充：若要替換成完整真實球員名單，請用同樣結構覆寫 `js/data/players_mlb.js`：
```js
const MLB_PLAYERS = [
  {"name":"Shohei Ohtani","team":"Los Angeles Dodgers","pos":"DH/OF","salary":70000000},
  ...
];
```
